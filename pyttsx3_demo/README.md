# 🔊 pyttsx3-demo
pyttsx3를 이용한 오프라인 기반 음성 합성 데모. 

## ✅ 사용 기술
- Python
- [pyttsx3 (Text-to-Speech for Python)](https://pypi.org/project/pyttsx3/)

## 🚀 실행 방법
1. Python 3.10 가상환경 생성
    `python -m venv pyttsx3_env`
2. 가상환경 활성화
    `.\pyttsx3_env\Scripts\activate`
3. 필요한 패키지 설치
    `pip install pyttsx3`
4. 실행
    `python pyttsx3_test.py`
