# 🔊 gtts-demo
Google Text-to-Speech(gTTS)를 이용한 파이썬 기반 음성 합성 데모.

## ✅ 사용 기술
- Python
- [gTTS (Google Text-to-Speech)](https://pypi.org/project/gTTS/)

## 🚀 실행 방법
1. Python 3.10 가상환경 생성
2. 필요한 패키지 설치:
    1) gTTS 설치
    `pip install gTTS`
    2) new pip 설치 필요시
    `python.exe -m pip install --upgrade pip`
3. 실행
    `python gtts_test.py`
